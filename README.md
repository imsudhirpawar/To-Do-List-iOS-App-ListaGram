# To-Do-List-iOS-App-ListaGram

This is a simple to-do list app built for iOS using SwiftUI.

## Features

The app allows users to:

- Add tasks to their to-do list
- Edit existing tasks
- Mark tasks as complete
- Delete tasks from the list
